### Goal?

"NN Compiler : Original Model -> Compiled Model"

Omodel = compile(Imodel)

To show that 'compile' is semantic-preserving, it is necessary to show that ``Omodel ~= Imodel``

Q. How to ``Omodel ~= Imodel``?

```
For any 'inference_input'

inference_output_from_Omodel = Omodel(inference_input)
inference_output_from_Imodel = Omodel(inference_input)

inference_output_from_Omodel ~= inference_output_from_Imodel
```
