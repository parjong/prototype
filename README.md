# prototype
For fast prototyping!

## Goal

Let's find how to use telegram for bot communation!

## How to run
1. Set the `BOT_TOKEN` environment variable with your telegram bot token.
   ```bash
   export BOT_TOKEN=<your bot token>
   ```
2. Run the bot.
   ```bash
   python -m src.telegram_bot
   ```
